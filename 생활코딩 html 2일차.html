HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. Web browsers receive HTML documents from a web server or from local storage and render them into multimedia web page, HTML describes the structure of a web page semantically and originally included cues for the appearance of the document

HTML elementes are the building blocks of HTML pages. With HTML constructs, imanges and other objects, such as interactive forms, may be emvedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, linksm quotes and other items, HTML elements are delieneated by tags, written using angle brackets.
